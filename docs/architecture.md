# 架构设计文档

## 系统架构概览

本文档详细说明了 ConceptGraph AI 的系统架构设计。

### 架构图

```
┌─────────────────────────────────────────────────────────────┐
│                    前端层 (Frontend Layer)                     │
│         React + D3.js/Echarts + WebSocket                     │
└─────────────────────────────────────────────────────────────┘
                            ↓ HTTPS/WSS
┌─────────────────────────────────────────────────────────────┐
│                  API Gateway (Nginx/Traefik)                  │
└─────────────────────────────────────────────────────────────┘
                            ↓
        ┌───────────────────┼───────────────────┐
        ↓                   ↓                   ↓
┌──────────────┐  ┌──────────────────┐  ┌──────────────┐
│ 概念查询服务  │  │  知识图谱服务     │  │  可视化服务   │
└──────────────┘  └──────────────────┘  └──────────────┘
        ↓                   ↓                   ↓
┌──────────────────────────────────────────────────────────┐
│              智能体编排层 (Agent Layer)                     │
│  ┌────────────┐  ┌────────────┐  ┌──────────────┐       │
│  │ 关联挖掘    │→ │ 知识校验    │→ │ 图谱构建      │       │
│  │ Agent      │  │ Agent      │  │ Agent        │       │
│  └────────────┘  └────────────┘  └──────────────┘       │
└──────────────────────────────────────────────────────────┘
        ↓                   ↓                   ↓
┌───────────┐  ┌────────────────┐  ┌──────────────────┐
│  LLM服务   │  │  向量数据库     │  │   图数据库       │
│ (OpenAI)   │  │  (Milvus)      │  │   (Neo4j)        │
└───────────┘  └────────────────┘  └──────────────────┘
        ↓                   ↓                   ↓
┌──────────────────────────────────────────────────────────┐
│            基础设施层 (Infrastructure)                      │
│  Docker + Kubernetes + Redis + MinIO + Prometheus         │
└──────────────────────────────────────────────────────────┘
```

## 数据流向

用户请求 → API Gateway → Backend → Agent → Neo4j → Frontend

## 云原生组件说明

- **Docker**: 容器化部署
- **Kubernetes**: 服务编排和自动扩缩容
- **Redis**: 缓存层
- **Neo4j**: 图数据库
- **Prometheus**: 监控
- **Grafana**: 可视化
