# ConceptGraph Frontend

React + TypeScript + D3.js 跨学科知识图谱前端应用

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
```

## 技术栈

- React 18
- TypeScript 5
- D3.js 7 (力导向图可视化)
- Ant Design 5 (UI组件库)
- Axios (HTTP客户端)

## 环境变量

创建 `.env` 文件：

```
REACT_APP_API_URL=http://localhost:8000/api/v1
```

## 功能特性

- 概念挖掘搜索
- 交互式知识图谱可视化
- 节点拖拽和缩放
- 节点详情展示
- 学科分类着色
- 可信度可视化
